2. Utwórz projekt → Realtime Database → Utwórz bazę
3. Reguły: zmień na ".read": true, ".write": true
4. Skopiuj URL bazy (np. https://xxx-rtdb.firebaseio.com) i dodaj /data.json
5. Wklej tutaj i kliknij Zapisz URL
